Frequently asked questions

Is BurnCloud reliable like OpenRouter?+
Yes. BurnCloud routes to the same official provider APIs with smart load balancing and 99.9% uptime. Because it's open source you can also self-host for full control.
Does BurnCloud have all the same models?+
BurnCloud supports all major frontier models plus deeper Chinese model coverage (DeepSeek, Qwen, Doubao, MiniMax, GLM, Kimi, Step) than most OpenRouter listings.
How do I verify BurnCloud is safe?+
The gateway is open source on GitHub. You can read the routing code, confirm there's no request logging, and reproduce the running build hash yourself.
What if BurnCloud goes down?+
BurnCloud is a single open-source binary you can self-host in under 10 minutes, so you're never locked to our hosted service.
Can I use both OpenRouter and BurnCloud?+
Yes. Both use the OpenAI-compatible API, so you can route different workloads to each by swapping the base_url and key.
