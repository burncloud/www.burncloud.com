One advantage of using BurnCloud is the ability to switch between Claude and GPT-5 with a single API key. Here's how:

# Initialize client

    api_key="your-burncloud-api-key",
    base_url="https://api.burncloud.com/v1"
)

# Use Claude Sonnet 4.6 for coding tasks

    model="claude-sonnet-4-6",
