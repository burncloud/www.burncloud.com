Quick setup (5 minutes)

  1. Create a BurnCloud account at ai.burncloud.com — sign up with email, no credit card.
  2. Add credits — minimum $5 via Stripe, credits never expire.
  3. Get your API key — Dashboard → API Keys → Create.
  4. Make your first call with the code below.
import openai

client = openai.OpenAI(
    base_url="https://api.burncloud.com/v1",
    api_key="your-burncloud-key"
)

response = client.chat.completions.create(
    model="deepseek-v4",
    messages=[
        {"role": "system", "content": "You are a helpful assistant."},
        {"role": "user", "content": "Explain how APIs work"}
    ],
    max_tokens=1000
)

print(response.choices[0].message.content)
import OpenAI from "openai";

const client = new OpenAI({
  baseURL: "https://api.burncloud.com/v1",

const response = await client.chat.completions.create({

});

console.log(response.choices[0].message.content);

Advanced usage

Once your first call works, two patterns cover most production needs: streaming for responsive UIs, and function calling for tools and agents.

Streaming responses

for chunk in client.chat.completions.create(
    model="deepseek-v4",
    messages=[{"role": "user", "content": "Hello"}],
    stream=True
):
    print(chunk.choices[0].delta.content or "", end="")

Function calling

tools = [{
    "type": "function",
    "function": {
        "name": "get_weather",
        "parameters": {
            "type": "object",
            "properties": {
                "location": {"type": "string"}
            }
        }
    }
}]

Yes. Via BurnCloud, your requests go directly to DeepSeek's official API. BurnCloud is open source — you can verify there is no logging. Read our security policy →
How does DeepSeek V4 compare to ChatGPT?+
DeepSeek V4 performs comparably to GPT-5.4 on most benchmarks at roughly 17x lower cost. For most use cases, you won't notice a quality difference.
